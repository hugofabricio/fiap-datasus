# BuscaSUS - API

1. **Instale as dependências.**

   ```shell
   npm install ou yarn install

   ```

2. **Configurar variáveis de ambiente.**

   ```
      PORT=5000
      DB_URI="mongo_uri"
   ```

3. **Rodar em modo de desenvolvimento.**

   ```shell
   npm run start:dev ou yarn start:dev
   ```
